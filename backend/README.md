======================How to set up and run the backend (including any installation steps)==========================================

Setup
>>create a project folder and place the index.js file and the controllers, models and routes folders
>>initial node in your project folder by running npm init y in your project folder , this will initialize the node with in the root folder

=====================installing system dependencies =======================================================
>>open the terminal and navigate to the root folder
>>run this command to install dependencies -  npm install body-parser dotenv express mongoose


=====================database================================================================================

>> the database is an online database that is MongoDB
>> the access link is there in the .env file of the project folder so the database is accessed through that link

>>if you want to use another mongo db vist https://www.mongodb.com/ and follow the steps to create and link your database to the server
